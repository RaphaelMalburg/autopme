# Nichos — visão geral e prioridade

Cada nicho tem um guião: perfil, dores, automação a vender, exemplo de ROI e
abordagem. Há um [preset](../presets/) por nicho para carregar a demo em segundos.

## Prioridade de ataque
**Vaga 1 — ROI óbvio (faltas/mensagens), ciclo curto, fácil de demonstrar:**
| Nicho | Dor nº1 | Automação wedge | Ficheiro |
|---|---|---|---|
| Clínicas dentárias | No-shows | Confirmações + lembretes | [clinicas-dentarias.md](clinicas-dentarias.md) |
| Estética & beleza | Marcações manuais, inativos | Confirmações + reativação | [estetica-beleza.md](estetica-beleza.md) |
| Ginásios & estúdios | Abandono, dúvidas repetidas | Reativação + FAQ | [ginasios-estudios.md](ginasios-estudios.md) |
| Restaurantes | Reservas em horas de pico | Reservas + FAQ | [restaurantes.md](restaurantes.md) |
| Oficinas | Pedidos dispersos, sem aviso | Triagem + "viatura pronta" | [oficinas.md](oficinas.md) |
| Imobiliárias | Leads frias, resposta lenta | Captação + visitas | [imobiliarias.md](imobiliarias.md) |

**Vaga 2 — valor alto mas ciclo mais longo / triagem:**
| Nicho | Dor nº1 | Automação wedge | Ficheiro |
|---|---|---|---|
| Advocacia | Triagem manual de pedidos | Qualificação + agendamento | [advocacia.md](advocacia.md) |
| Contabilidade | Esclarecimentos repetidos | FAQ + recolha de docs | [contabilidade.md](contabilidade.md) |
| Farmácias | Interrupções ao balcão | Reservas + levantamentos | [farmacias.md](farmacias.md) |
| Padarias/Pastelarias | Encomendas sem processo | Registo de encomendas | [padarias.md](padarias.md) |
| Alojamento local/Hotéis | Pedidos multilíngues | Reservas + FAQ multilíngue | [alojamento.md](alojamento.md) |

## Regra
Escolhe **1 nicho** e domina-o antes de abrir o próximo. Casos de sucesso num nicho
vendem o nicho inteiro (mesmas dores, mesma demo, mesmo ROI).
