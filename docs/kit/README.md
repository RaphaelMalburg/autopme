# Kit AutoPME Porto — vender, montar e entregar automação/IA para PMEs

Tudo o que precisas para **prospectar, vender, instalar e entregar** automação de
WhatsApp, voz e processos a pequenas e médias empresas no Porto. Organizado por
função. Lê pela ordem dos números na primeira vez.

## O caminho mais curto para o primeiro cliente
1. Lê [estratégia](vendas/01-estrategia.md) e escolhe **1 nicho** (sugestão: clínicas dentárias).
2. Prepara a demo com o [preset](presets/) desse nicho (carrega em 10 segundos).
3. Marca um **diagnóstico gratuito** com o [script de abordagem](vendas/03-scripts-venda.md).
4. Na visita: corre a demo → mostra ROI → entrega o [one-pager](vendas/one-pager-modelo.md).
5. Fecha **1 automação** (confirmações WhatsApp) com o [preço de lançamento](vendas/02-precos-pacotes.md).
6. Instala seguindo o [onboarding](whatsapp/onboarding-cliente.md).

> Princípio que atravessa tudo: **vende resultados, não tecnologia.** "Menos faltas e
> zero mensagens sem resposta" — nunca "agente de IA".
