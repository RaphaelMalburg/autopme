# Guião de Demo (passo-a-passo)

Como conduzir a demo no dashboard, em ~10 minutos, para qualquer nicho. Treina 2-3 vezes.

## Antes da visita (2 min)
1. Abre o dashboard e faz login.
2. Escolhe o **nicho** e clica **"Carregar exemplo do nicho"** (ou pesquisa o prospect real).
3. Clica **"Carregar demo"**. Está tudo pronto (agente, diagnóstico, WhatsApp).

## Durante a visita (a sequência que fecha)
1. **Abre pela dor** (30s): "Quantas faltas têm por semana? Quanto perdem com isso?"
2. **Voz — o wow** (2 min): "Ligar para o agente" → deixa o cliente ouvir o agente a
   responder como o negócio dele. *Não vendas isto; é para impressionar.*
3. **WhatsApp — a venda** (2 min): mostra o agente a responder no WhatsApp (FAQ +
   confirmação). "É isto que vai ter a trabalhar para si todos os dias."
4. **Diagnóstico + ROI** (3 min): "Gerar diagnóstico" → mostra dores, quick wins e o
   **número do ROI** no ecrã. Abre o talk-track se surgir objeção.
5. **Proposta** (2 min): "Proposta (PDF)" → gera o documento com marca e entrega/envia.
6. **Fecho:** "Começava por [automação]. Setup [€], [€]/mês. Avançamos?"

## Depois (1 min)
- "Enviar para Pipeline (Notion)" → o prospect entra no funil com o resumo.
- Define o próximo contacto.

## Dicas
- Usa o **preset** se não tiveres dados do prospect — a demo fica instantânea.
- Se a net falhar, a voz tem fallback de browser; o WhatsApp mostra na mesma o conceito.
- Mantém o ritmo: voz (wow) → WhatsApp (real) → ROI (número) → proposta (fecho).
- Nunca mais de 12 min. O objetivo é o **próximo passo**, não explicar tudo.
